# 2026-06-20 — About 페이지(국/영) + 전역 OG 공유 카드 (외부 소개 준비)

외부 커뮤니티(HN/GeekNews/디스콰이엇/Reddit) 소개 전, 첫인상·공유 미리보기를 다듬은 작업. 원래 UI/UX 작업목록 4개 중 우선순위 높음 2개(About, OG)를 처리.

## 기능

- [x] **About 페이지 국/영** — 독립 `/about/`(`content/about.md`) + `/about-en/`(`content/about-en.md`). AI 1인칭 자동 회고 컨셉·발행 파이프라인을 비개발자도 이해하게 설명. `hugo.toml [[menu.main]]`로 상단 메뉴 About 노출, 두 페이지 상호 링크. 영문은 "글 본문은 한국어"임을 정직히 명시(외부 소개용). 영향: `content/about.md`·`content/about-en.md`·`hugo.toml`
- [x] **발행 파이프라인 다이어그램 국/영** — 기록 소스(docs/todo+ts)→Claude Code 작성→GitHub 빌드·공개 흐름을 하우스 스타일 `.drawio`→헤드리스 PNG. 영향: `docs/assets/devlog-publish-pipeline{,-en}.{drawio,png}`·`static/img/`
- [x] **img shortcode** — baseURL(`/devlog/`)을 `relURL`로 안전히 붙여 마크다운 이미지 임베드(goldmark `unsafe` 안 켜고). 영향: `layouts/shortcodes/img.html`
- [x] **전역 OG/트위터 공유 카드** — `static/og.png`(1200×630, 보라 그라데이션) + `hugo.toml [params] images`. 프로덕션 빌드서 `og:image`·`twitter:card=summary_large_image`·`twitter:image` 출력 확인. 카드 소스·재생성: `tools/og/card.html`+`render.sh`(헤드리스 chromium). 영향: `static/og.png`·`hugo.toml`·`tools/og/`

## 버그수정

- [x] **국문 About→영문 링크 상대경로 오류** — `[..](about-en/)`이 트레일링슬래시 디렉토리(`/devlog/about/`) 기준 `/devlog/about/about-en/`로 404. `../about-en/`로 수정. 프로덕션 빌드 산출 검증 중 발견. 영향: `content/about.md`

- [x] **글 본문 영문화는 보류** — About만 KR/EN, 글은 한국어 유지. HN/Reddit엔 영문 About+메타스토리로 소개(컨셉은 글이 한국어여도 전달). 본격 진행 시 **자동 회고 루틴(devbox `vm/devlog-auto-retro.sh`)이 EN도 생성**해야 지속됨(안 하면 백로그만 EN, 이후 stale). Hugo i18n+백로그 ~20편 번역 필요. 영향: 결정만(코드 변경 없음)
